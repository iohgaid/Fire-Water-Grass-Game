# 🔥💧🌿 Fire Water Grass Game

## Overview

Fire Water Grass is a simple browser-based game inspired by Rock Paper Scissors.

### Rules

* 🔥 Fire beats 🌿 Grass
* 💧 Water beats 🔥 Fire
* 🌿 Grass beats 💧 Water
* Same choices result in a tie

The first player to reach **5 points** wins the game.

## How to Run

1. Download or clone the project.
2. Ensure the following files are in the same folder:

   * `index.html`
   * `style.css`
   * `script.js`
3. Open `index.html` in any modern web browser.

No installation or additional dependencies are required.

---

## How to Play

1. Click **Start Game**.
2. Choose one of:

   * Fire
   * Water
   * Grass
3. The computer will randomly make its choice.
4. The result of the round is displayed.
5. Scores are updated automatically.
6. The first player to reach **5 points** wins.
7. Click **Restart** to play again.
